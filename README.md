# 🛍️ Vrinda Store Annual Sales Data Analysis (2022)

# Project Overview
This project presents a comprehensive **Exploratory Data Analysis (EDA)** of Vrinda Store's 2022 annual sales data using **Microsoft Excel**. The goal is to uncover key business insights — including top-performing states, customer demographics, order channels, and product categories — to help the store strategize and boost sales in 2023.

## 🎯 Objective
"Vrinda Store wants to create an annual sales report for 2022 so that they can understand their customers better and grow more sales in 2023."

## 📊 Key Business Questions Answered
- Which month had the highest sales and orders?
- Who purchased more — Men or Women?
- What were the different order statuses in 2022?
- Which are the Top 10 states contributing to sales?
- What is the relation between Age, Gender, and Number of Orders?
- Which sales channel contributed maximum revenue?
- Which product category was the highest selling?

## 🛠️ Tools & Techniques Used
| Tool | Purpose |
| Microsoft Excel | Data Cleaning, Processing, Analysis & Dashboard |
| Pivot Tables | Aggregating and summarizing data |
| Charts & Graphs | Visual representation of trends |
| Slicers | Interactive filtering on the dashboard |
| Conditional Formatting | Highlighting key data points |

## 🔍 Data Cleaning Steps
- Removed duplicate and null values
- Standardized inconsistent entries (e.g., "M"/"Men", "W"/"Women")
- Converted data types for date and numerical columns
- Created new derived columns (Month, Age Group)

## 📈 Key Insights
- 🚺 **Women** are more likely to buy (~65%) compared to men
- 🏆 **Maharashtra, Karnataka & Uttar Pradesh** are the Top 3 states (~35% of sales)
- 👩 **Adult women (30–49 age group)** contribute the most (~50%)
- 🛒 **Amazon, Flipkart & Myntra** are the top 3 channels (~80% of orders)
- ✅ ~92% of orders were successfully delivered

## 💡 Final Recommendation-
**Target women customers aged 30–49 in Maharashtra, Karnataka & Uttar Pradesh** by showing ads, offers, and coupons on Amazon, Flipkart & Myntra.
